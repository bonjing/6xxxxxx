Release Date  :  21/01/2012                Protection  :  G4WL
Release Type  :  RIP 21x5.00MB             Company     :  WB Games

Release Notes:

The Mortal Kombat Arcade Kollection, contains all three classic games
(Mortal Kombat, Mortal Kombat 2, and Ultimate Mortal Kombat 3)
available for download.  All three games are complete versions of
their original and can now be played online as well.

Key differences: leaderboards, online matches as well as Achievement
and Trophy support.

RIPPED: Nothing

Install Notes:

1. Unpack the release with WinRAR or equivalent
2. Install by running unl-mkak.exe
3. Start game with MKHDGame.exe in \BINARIES\WIN32\
4. Enjoy!

autoselect the language you got Games For Windows Live intalled in.
