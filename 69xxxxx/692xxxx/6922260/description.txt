1.)Unrar file folder to desktop.

2.)Open Folder

4.)Move all the files to your /Windows/Fonts directory.

5.)Copy and overwrite files....yes you will have errors, some fonts are already installed and some may be to old for Windows Seven to use.

6.)Delete empty folder on desktop, Done
