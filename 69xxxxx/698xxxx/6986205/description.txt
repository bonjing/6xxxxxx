This software tool allows you to adjust the CPU frequency (overclock) right from Windows without having to reboot or access BIOS. Very useful for laptops.

This is latest version that doesn't request a key from "impolite users".
 
http://www13.plala.or.jp/setfsb/ - homepage and clock generator list. You should find one that suits your laptop or motherboard.
