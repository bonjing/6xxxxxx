Emsisoft Emergency Kit contains a set of free programs that can be used without installing scan and disinfect an infected computer from malicious software such as viruses, Trojans, Internet worms, spyware and adware. Your emergency kit for infected computers detects and removes malicious programs (more than 4 million known threats). 100% portable - ideal for USB-stick. Included HiJackFree and BlitzBlank. The main components of Emsisoft Emergency Kit

Detailed description:

Emsisoft Emergency Kit Scanner
Powerful anti-virus scanner Emsisoft Scanner GUI. Search on the infected computer viruses, thrones, spyware, adware, worms, dialers, keyloggers and other malicious software.

Start Emsisoft Emergency Kit Scanner, double clicking on the file a2emergencykit.exe. Found malicious programs can be moved to quarantine or permanently removed.

Emsisoft Commandline Scanner
This scanner has the same function as the Emsisoft Emergency Kit Scanner, but without a GUI. Anti-virus scanner from the command line for experienced users and can be used for batch jobs.

To start Emsisoft Commandline Scanner, follow these steps:
- Open a command prompt (Start: cmd.exe)
- Navigate to the USB-flash drive (for example: f), and then to the folder of executable files (eg: cd run)
- Start the scanner by typing: a2cmd.exe
Next you will see a page with a description of all available options.

Emsisoft HiJackFree
HiJackFree helps advanced users to detect and remove malware manually. With HiJackFree you can manage all the active processes, services, drivers, startup, open ports, host file entries and many others. The component is designed for complete control over the system.

Emsisoft BlitzBlank
BlitzBlank - a tool for advanced users and those who deal in malicious software on a daily basis. Malware infections are not always easy to clean - Malicious software uses clever techniques to protect itself from deletion. In most cases, often almost impossible to remove malicious program file, while running Windows. BlitzBlank remove the files, registry entries and drivers to load Windows and other programs.

Installation:
Extract the contents of Emsisoft Emergency Kit on a USB-stick and make your own versatile tool to scan and clean an infected computer.
