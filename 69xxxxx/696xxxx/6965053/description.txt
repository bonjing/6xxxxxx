Kaspersky TDSSKiller 2.7.5.0


Kaspersky TDSSKiller - A tool to detect and remove rootkits, malicious programs that secretly introduced into the system and hooks system funktsii.Kaspersky TDSSKiller designed to fight malware family Rootkit.Win32.TDSS (Tidserv, TDSServ, Alureon); bootkits; rootkits.

Features:
- The utility has a graphical user interface.
- The utility supports 32-bit and 64-bit operating systems.
- Tool can be run in Normal and in Safe Mode.
- The utility runs without installation (Portable)
