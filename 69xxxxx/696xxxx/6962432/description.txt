DVDFab All-In-One:
==================

DVDFab All-In-One completely includes all individual DVDFab apps -- 2D to 3D Converter, DVD Copy, DVD Ripper, Blu-ray Copy, Blu-ray Ripper, Blu-ray to DVD Converter, Blu-ray 3D Ripper, Video Converter, and File Transfer -- so as to provide the universal solution to all DVD/Blu-ray/video issues.


Just run the installer and you're good to go!!

If you want to update this release, make a copy of msvcr90.dll in the install folder before updating.
After update, copy it back into the install folder replacing the original!
