iRamDisk uses a block of memory to creates a virtual RAM Volume displayed like real disk by Mac OS X. Data into memory are written and reader more quickly than the system disk. Move the temporary files into RamDisk may reduce the delay of a process (compilation, image processing,...). iRamDisk is based on Unix common

System Requirements

Intel Mac, OS, X.6 or greater
