iDeveloper - Icon Generator is a simple application which makes your life as an iOS developer or designer a bit easier. This app generates all the required icons for both iOS and Android. The only thing you need to provide is an 512x512 icon artwork and the Icon Generator generates all the nessesary icons with the correct names and sizes for you without any loss of quality.

FEATURES

* Very easy to use with slick interface design.

* Preview icons with and without rounded corners & gloss.

* Generate icons for iPhone, iPad, Android or all.

* Generate all required icons for your iOS and/or Android application.

* Save generated icons on disk.

ADDITIONAL INFORMATION

* Your source icons must be at least 512x512 pixels and square.

* This application only generates square images. (iOS icons need to be
  square)

* Supported/Tested formats: JPG, JPEG, PNG, GIF, PSD and Ai.
