"Crypto World" is a easy application for encrypting and decrypting text.
The program is very light and easy to use for everybody,
you can use it on every windows based computer.

Version 1.0 - 14-01-2012
- First release
- Encrypt text
- Decrypt text

Version 1.1 - 15-01-2012
- Password protected encrypt system.
- You need the password to decrypt the text.
- MD5 Conversion tool (for password, names etc.)
- A webpage for Crypto World. (http://progs.friendsinwar.com/prog_crypto_world.php)
- Help File for better understanding the program.
