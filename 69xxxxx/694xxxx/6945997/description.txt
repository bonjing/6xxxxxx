Tweaks en aanpassingen:
***********************
Er zijn 2 aanpassingen doorgevoerd aan de standaard instellingen, om het geheel goed te laten werken.

1. UAC het beruchte gebruikers account beheer is uitgeschakeld.
is nodig voor het installeren van Unattended/Silent installers.

via start - msconfig - hulpprogramma's - UAC instellingen. weer aan te zetten.

2. Energieschema is op Hoge prestatie gezet.
omdat anders je computer in stand-by gaat na een tijdje.

via bureaublad eigenschappen - schermbeveiliging energie-instellingen wijzigen -
selecteer een energiebeheerschema en dan vervolgens op Gebalanceerd (aanbevolen) zetten.

Windows 7 Loader:
*****************
normaal wordt deze verwerkt in de installatie, deze keer heb ik ervoor gekozen dit niet te doen zodat je een eigen code kan gebruiken.

wil je toch de Loader installeren:

ga met windows verkenner naar de dvd of usb-stick open de map Windows Loader v2.0.9
en start Loader.exe op. en druk op install.
