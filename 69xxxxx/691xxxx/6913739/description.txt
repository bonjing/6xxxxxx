SILENT HILL

Custom game for the sony ericsom Xperia Play

Instructions:

1- Copy the file com.sony.playstation.SLUS_00707.apk to anywhere on the sd card of your Xperia Play.

2- Copy the folder named com.sony.playstation.SLUS_00707 to your sd card Android/data folder.

3- On your Xperia Play, install the game using the file com.sony.playstation.SLUS_00707.apk.

Observations:

- Works perfectly on Xperia Play with Android 2.3.4
- May presents some graphics problems with alpha textures.

Enjoy.
