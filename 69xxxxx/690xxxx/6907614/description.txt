How to install Aura Fate of The Ages:

1.Mount with daemon tools 4.08 or alcohol 120% or burn 3 files .iso
2.Launch installation
3.Apply patch 1.4 IN GAME DIRECTORY
4.Replace exec file in game directory with Aura1.exe in CURA folder
5.Run the game
6.If you need it, there's the Walkthrough of the game in "soluzione" folder
