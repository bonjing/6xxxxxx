FlashFXP is the premier FTP, FTPS, SFTP, FXP file transfer program used to transfer files in an intuitive interface. FlashFXP provides an easy to use interface for common file transfer tasks and an optional tree view for easy directory navigation. 

 Features:
 Queue multiple files for uploading and downloading.
 Session recovery - Recover lost transfer queues due to power outages or system crashes.
 Server to server transfers (FXP)
 SFTP support
 IPv6 support

 FTP/HTTP proxy
 SOCKS 4/5 support
 Remote file editing with automatic upload on save.
 Search for files
 ...and much much more!
