Description: 
------------ 
This program will help you design and print a professional business card. The program's interface has been designed so that even a beginning user will not have problems using the program. 

The design process is based on the placement of elements accessible in the program such as figures, images and backgrounds, and freely shifting them on the prepared project. Another advantage of the program is the ability to save data in a company or people so that it can be used later while designing a business card. The program uses templates which you can prepare on your own, or you can use ready models, templates, access cards in it. There are 400 examples of business cards in the program.

- A dozen or so effects which can be used while working on all objects (text, image, backgrounds) for example: shadows, textures, sepia, gradients and other

- Print on both sides of the paper (duplex) 
- Multi
