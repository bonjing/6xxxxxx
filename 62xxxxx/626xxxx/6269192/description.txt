HOW TO INSTALL:

Extract to %appdata%/.minecraft , or C:\Users\Logan2\AppData\Roaming\.minecraft 
and run Minecraft.exe. When logging on, enter a random username and password, then click 'Play Offline.'

1) Extract to C:\Users\Logan2\AppData\Roaming\.minecraft

2) Run Minecraft.exe

3) Type in a random username and password

4) Click 'Play Offline.'

-------------------------------
HOW TO PLAY ON MULTIPLAYER SERVERS:
You can only play on servers that have online-mode and verify-names set to false.
