NX, also known as NX Unigraphics, is an advanced CAD/CAM/CAE software package developed by Siemens PLM Software.

[url]http://en.wikipedia.org/wiki/NX_(Unigraphics)[/url]


How to install:

-Mount or burn the ISO file.
-Install NX7.
-Do not install licensing.
-Copy files according to paths inside.
-Put nx7.lic to "Program Files\UGS\NX 7.0\UGFLEXLM" and setup/start license server there using lmtools.exe
