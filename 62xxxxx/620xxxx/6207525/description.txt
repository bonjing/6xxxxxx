ALZip v8.12.0.3


ALZip is a file compression program that can unzip 40 different zip file archives. ALZip can zip files into 8 different archives such as ZIP, EGG, TAR, LZH and others. In ALZip version 8, ALZip's own EGG format, that Unicode and other powerful functions support, can be used.

ALZip is compatible with Windows 2000, XP, Vista(32/64bit), and 7(32/64bit). Support for the following compression formats is included:

ZIP, 7Z, RAR, EGG, TAR, TBZ, LZH, Z, JAR, ALZ, ACE, ARC, ARJ, B64, BH, BHX, BIN, BZ, BZ2, CAB, EAR, ENC, GZ, HA, HQX, ICE, IMG, LHA, MIM, NRG, PAK, TBZ2, TGZ, UUE, UU, WAR, XXE, ISO, LCD, ZOO

ALZip can zip to the following formats:

ZIP, EGG, ALZ, TAR, TBZ, TGZ, LZH, JAR

ALZip can also:

*Restore corrupted files
*Span a zip file into zip files
*Merge spanned files into one
*Set password using AES-128bit/AES-256bit encryption
*Check for viruses using your anti-virus software
*Create self-extracting archives (EXE) to let people open your archive with no file compression program installed
*Check for errors in zip files
*Process command line functions for expert users
