All I did was use WAIK to import each version into 1 install.wim file and recreated the .iso. These installs will be the original Windows 7 install of your flavor!

I have also included the repair module for BOTH x86 and x64.

This will fit on a DVD5 but WILL NOT work on a 4 gig USB stick. It will work on an 8 gig USB stick though.

I used the following MSDN Original ISO\'s to create this 11-in-1 AIO. Nothing is changed or added to the actual Windows install. The only thing that was added was to create a repair module for x64!

Use Daz\'s Loader (Currently 1.9.7) to activate all but Enterprise Versions.

Use KMS Activator (Currently 5.0, but has Alpha 6.0 out) to activate Enterprise Versions.
