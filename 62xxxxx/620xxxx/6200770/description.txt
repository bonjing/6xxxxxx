Xbox 360 Windows Driver v1.2


This software driver compatible with Windows XP or later allows your Xbox 360 controller to be used as a compatible game-pad for the Microsoft Windows Operating System. This works with both wired and wireless controllers, but in order to set up your wireless controller you will need this product:
 
http://www.amazon.com/Xbox-Wireless-Gaming-Receiver-Black/dp/B0032A0RBC/ref=sr_1_3?ie=UTF8&qid=1298603341&sr=8-3

The rest is pretty self-explanatory. Enjoy!
