Dreamweaver CS5 Getting Started with HTML5
Author: Joseph Lowery
In Dreamweaver CS5: Getting Started with HTML5, author Joseph
Lowery explains how to develop cross-browser compatible web
sites utilizing the enhanced capabilities of the HTML5
specification. The course details Dreamweaver's support for
HTML5, including templates, additional code hints, and a
multi-screen preview for mobile device development. Tutorials
on embedding native audio and video players without plug-ins
incorporating non-standard fonts with @font-face, and creating
forms with spinners, sliders, and calendars are also included
Exercise files accompany the course

Topics include:

* Updating Dreamweaver CS5 to support HTML5
* Developing for mobile devices and tablets
* Understanding enhanced structural tags
* Creating a basic page layout
* Drawing with the canvas tag
* Implementing CSS3 type effects
* Setting up styles
