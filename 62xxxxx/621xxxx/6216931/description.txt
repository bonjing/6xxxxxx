WBFS Manager v1.2 for MacOS X


+-----------------------+
|Application Description|
+-----------------------+

This tool can be used to format a hard drive with a WBFS partition, transfer Wii ISO images directly to a hard drive, manage games on WBFS formatted hard drives and download cover images.

+------------+
|Instructions|
+------------+

This software is designed for Macintosh OS X 10.4 or later. The application should be self-explanatory, but if you have any questions just post them here and I'm sure a Mac User can answer you promptly.
