An Old Naval War Game.  If you want info about it use a search engine.  The File is in .iso.  If you are wondering how to use this on an emulator, download winrar, right click the iso image and extract files.  Once extracted, you'll find one called SLUS.xxx.xx  .  Replace the last two SLUS.xxx.*xx* with SLUS_xxx.elf.  Go to pcsx2.net and find all the info there. Have Fun.
