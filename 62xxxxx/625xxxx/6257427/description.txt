Game notes

Shank immerses gamers in a side-scrolling, highly cinematic revenge story.
With the ability to string together endless combos of mayhem and destruction
and an array of blood-spilling weapons at your disposal, Shank seeks to reboot
the 2D brawler and take it to a level of production never before seen.

General trainer information

Make sure to run the trainer as administrator.
Before you are able to use any of the shown options below. You need
to press F12 first to enable the trainer.

Ingame hotkeys

#    Hotkey              Option
F12                 Enable Trainer

#    Hotkey              Option
1    F1                  Toggle Infinite Grenades
2    F2                  Toggle Infinite Health
3    F3                  Toggle Super Jump
4    F4                  Toggle One Hit Kill

Trainer notes

F1 - Infinite Grenades

Hit F1 and you will have an unlimited amount of grenades.

F2 - Infinite Health

Hit F2 to get an infinite amount of health.

F3 - Super Jump

With this option enabled you can jump much higher than usual.

F4 - One Hit Kill

When enabled you will be able to kill all your enemies with
one hit. Note that certain enemies need to be killed using
a special attack!. Those are shown with red indicators!.

Install Notes

1. Unpack the release with WinRAR or equivalent
2. Run the trainer from the desktop.
3. Start the game
4. Toggle desired Options on/off
