Installation:

1. Remove any previous patches installed.

2. Unrar files, run (setup.exe) and follow installer until it finishes.

3. When install finishes manager.exe will run, be sure that manager is attached to the exe file, if it shows errors run manager.exe as administrator.

4. Choose other 2D to be applied in the patch (exit to leave default nPower league).

Important: no konami updates needed.
