		Evolution Dekaron V7R1 Client

A [GM]Goro Release [3]

	This release is the third release in the [GM]Goro 
Release Series and distributes the client used to play 
Evolution Dekaron.

The Version 7 Release 1 (V7R1) is the latest in Evolution 
Dekaron (EDK) gaming, bringing you not only the long awaited
'Aloken' but also Action 9 featuring new maps and new armour 
the like of which haven't been seen before. 

With updated quests, fixed bugs and adjustments the V7R1 promises 
to be one of the most exciting releases to date. One of the 
key features of this installment is not only higher security 
but a lower file size.

A final exciting feature, is the addition of a second channel 
due to the ever increasing number of players. 


Full details can be found at  
http://www.evolutiongames.net/dekaron

A complete installation guide can be found here:  
http://evolutiongames.net/dekaron/forums/viewtopic.php?f=29&t=1764
