MicroType 3.0 (a revision of MicroType Multimedia) is an alphabetic, numeric, skillbuilding, and keypad program with motivating graphics, games, and a word processor with timer build in. This software program teaches correct finger placement and builds basic skills, then works on speed and accuracy.
