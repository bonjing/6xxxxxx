If you guys run into Username & Password Issue at Start UP do this,





This info is for anyone having probs with username and password.

First, boot into safe mode (f8) Go to control panel / user accounts / manage another account / create a new account.

Make sure you click Admin. Put your name in for \\\"new account name\\\" no password. Click create account. Restart. This way you have Admin permissions.



This err is not for installing the OS or within the OS but this is due to the time setting parameters.. !!
