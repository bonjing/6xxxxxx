This is software for the blind.  It is a screenreader that helps blind persons to use windows computers.

This is Jaws For Windows V12.0.1158 for 32bit Windows.
If anyone has the 64bit version, please let me know.

First install J12.0.1158-32bit.exe.
Then run pj12-32b1.exe to patch the necessary files.
And you should be good to go.

for more info on jaws for windows, visit http://freedomscientific.com/ or http://hj.com/
