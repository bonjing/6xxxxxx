This logical game will fascinate lovers of brain teasers and those who enjoy solving challenging puzzles. Although the first levels are relatively easy, each one grows progressively more difficult and develops the player's skills. Besides using logical abilities, the player will need quick reflexes. Sometimes it's not easy to catch a bonus block!

Brief description of the game rules:

In order to pass each level the player has to break all empty blocks trying to save filled ones. Each time the player breaks an empty block he gets several points. Each time he breaks a filled block he loses several points and a life. If the player breaks a bonus block, which appears from time to time during the game, he gets various advantages.
