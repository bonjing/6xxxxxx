Symantec Endpoint Protection 11.x for Macintosh automatically removes Mac and PC viruses from email attachments, Internet downloads, and other sources to safeguard the corporate network. Centralized deployment, administration, reporting, and distribution of virus definition updates simplify enterprise-wide management of Mac clients.

* If you are running any version of Norton AntiVirus, you must 
  uninstall it before installing Symantec Endpoint Protection 11.0 for 
  Mac.

* Do not install or run Symantec Endpoint Protection 11.0 for Mac from
  a UFS formatted disk.

* Do not install or run Symantec Endpoint Protection 11.0 for Mac from
  a Case-Sensitive HFS+ formatted disk.
