About:
Web Builder is an WYSIWYG (What-You-See-Is-What-You-Get) utility that gives you the possibility to build web pages. WYSIWYG means that the finished page will display exactly the way it was designed. The program generates HTML (HyperText Markup Language) tags while you point and click on desired functions; you can create a web page without learning HTML.
