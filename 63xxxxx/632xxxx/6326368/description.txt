LDR stands for Linux Done Right. It is an Arch Linux derrived GNU/Linux distro which has many software components pre-configured so you can just install and get on with it! 

Please note LDR is in it's very early stages of development.We would suggest you read our Todo list before continuing.
If you like what we have so far, please let us know, or offer to contribute!
