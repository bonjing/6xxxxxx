This VirualBox Image (vdi) includes:

-Ubuntu 11.04 installation
-Xampp server installation
-EyeOs web desktop environment installation
-Macbuntu Mac osx theme for ubuntu

This system is tested and working fine in localhost and in public network.
Use port forwarding and you can use this virtual server/desktop environment in public network. ( http://www.virtualbox.org/manual/ch06.html#natforward )
