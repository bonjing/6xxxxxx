XemiComputers Dynamic Info Screen v11.2.6.110512 Portable

Dynamic Info Screen is a program for creating digital signage solutions using standard PC computers running Windows. An info screen project consists of one or more scenes where each scene is made of different media objects showing various types of information. Completed info screen projects work as independent applications easily transferrable to any number of computers.
