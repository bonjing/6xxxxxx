Description
~~~~~~~~~~~

tlTerm (aka TshwaneTerm) is a fully internationalized, off-the-shelf software application for compiling terminology lists (termbase editor). tlTerm brings the power of the widely recognized TLex dictionary compilation software to terminology.
 
http://tshwanedje.com/terminology/
