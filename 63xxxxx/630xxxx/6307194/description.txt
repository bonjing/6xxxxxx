Release Description

Microsoft VirtualEarth Map Downloader is a cute tool that help you to get
small tile map images from Microsoft VirtualEarth. All downloaded small
images are saved on the disk. You can view downloaded maps by Map Viewer
Or you can combine them into a big BMP map. It supports proxy server. And
you can easily get the longitude and latitude of each downloaded small
tile image
