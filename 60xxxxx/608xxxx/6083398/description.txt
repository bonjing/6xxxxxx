Keyshot 1.9.54 X32
Works well under Windows 7 X32 Ultimate

Excellent for Solidworks 2010


HOW TO INSTALL:

1) Unrar everything on a new folder
2) Open \"setup keyshot_w32_1.9.54.exe\"
3) Install normaly - DO NOT RUN

4) Copy both KeyShot-Patch.exe and keyshot.lic to app dir \\bin, replace when asked.

5) Open KeyShot-Patch.exe ...and obviously patch


6) Right click on Keyshot.exe, select properties

On Compatibility, select \"Windows XP Service Pack 3\"

On privilege level, select \"Run This program as administrator\"
