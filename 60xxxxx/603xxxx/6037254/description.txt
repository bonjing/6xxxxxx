This game is a faithful remake of classic 89' action game - Wings Of Fury.

What's new in version 3.3:
- enhanced version (http://wingsoffury2.com/page/enhanced):

    * B25 bomber plane
    * 4 new levels
    * Rocket bunkers and ships
    * "Survival mode" - survive as long as you can


- new installer - all dependencies included
- gameplay fixes: no plane collisions on "easy" difficulty level, you can shoot while turning around, balancing
- graphic tweaks (night levels, HUD, explosion shake)
- new languages: French, Spanish (ES and AR)


Runs under Windows XP/Vista/7 (32/64 bit)

More info, help, forums @ http://www.wingsoffury2.com
