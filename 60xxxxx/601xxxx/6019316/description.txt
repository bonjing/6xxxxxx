Zygor's Leveling Guides 3.0.1554 for World of Warcraft Cataclysm.


-----------------------
Release Includes:
	Zygor Leveling Guides for both Alliance and Horde (Levels 1-85)
	In-Game Talent Build Advisor
	Only guide so far to cover both Worgen and Goblin starter zones
	Zygor Dailies and Event Guide for both Alliance and Horde

-----------------------
Fixes:
	Horde Guides Change Log:
	* Fixed an error that prevented the Northrend 70-72 guide from loading.
	Guides Viewer Change Log:
	* Changed default options to make guide more visible

-----------------------
Requirements:
	World of Warcraft (4.0.3a or higher)
	Zip program to unzip the download

-----------------------
Installation Instructions:
	1) Download
	2) Extract
	3) Copy extracted folder to your World of Warcraft/Interface/Addons folder
		Default Locations:
		- C:\\Program Files\\World of Warcraft\\Interface\\Addons (Win)
		- C:\\Users\\Public\\Games\\World of Warcraft\\Interface\\Addons (Win)
		- /Applications/World of Warcraft/Interface/Addons (Mac)
	4) Launch WoW.
	5) Once logged in and at your character screen, click the AddOns button and make sure that ZygorGuidesViewer and ZygorTalentAdvisor are checked.
	6) Launch the game and enjoy.


If you have any questions please post them here in the comments and someone should be able to help you.  If you experience errors with the addon either LUA errors or broken steps please be patient.  I am sure the Zygor team is working diligently to get a solid release.
