Autodesk SketchBook Pro 2011 - a program that offers advanced tools of conceptual drawing for designers and artists, regardless of the industry in which they operate. Designed specifically for the Tablet PC and Tablet PC, Autodesk SketchBook Pro 2011 provides all the tools necessary for the transition from pen and paper to a digital environment.

The program allows the user to create and save your own brushes, erasers, airbrushes, etc. These properties of brushes, such as size, transparency, roundness, slant and spacing of brush strokes are adjusted individually. In addition, as a form of brush strokes, you can specify any image, in which case the motion of the brush is put custom textures. The program interface is easy to learn, and those who have already mastered it can operate at high speed and maximum efficiency. Customizable menus provide easy access to key functions - for example, set the color and the choice of brushes. The canvas, which is drawing, you can resize, move and rotate. The program can generate from the user touches the straight lines and ellipses. Tools to specify the size of the brush is always at hand - to open the editor of brushes is not necessary. You can expand the capabilities of traditional painting, creating custom brushes to add texture.
