This is a collection of maps and necessary files for UT 99, aka UT GOTY.

There are 270 assault map (files), some of which may be "betas" or "previews", (I haven't vetted all of them).
There are also 20 other odd maps that came with the assault map packs I found.

They have been zipped into a single .rar file, which contains Map, Sounds, Music, Textures, and System folders with the filetypes from each map sorted into the appropriate folder already.

Simply copy the files from each folder to the appropriate folder in your Unreal Tournament directory, and it should work.


These maps have been collected from mainly from:
 
http://downloads.utassault.net/ 
http://thepiratebay.org/torrent/3507123/UT_Assault_Maps

And several other sources that aren't worth finding again.

But those two are by far the biggest, (and have an amount of overlap). Thanks guys!


The unzipped filesize is about 1.7 gigabytes. 


Although I will only list the map files here, all textures, sounds, music, and system files required to run them ARE included.
