Lynda.com Publisher 2010 Essential Training

In Publisher 2010 Essential Training, author David Rivers demonstrates how to create professional publications, such as brochures, newsletters, and menus. Using real-world examples, the course includes an overview of the different types of publications available in Publisher, shows how to use Publishers tools for modifying text, objects, and tables, and explains how to customize layout and design options. Tutorials on performing mail merges and preparing publications for the web and for print are also included. Exercise files accompany the course.

Topics include:

   * Exploring the user interface
   * Creating and saving publications
   * Adjusting page and print setup options
   * Working with bulleted and numbered lists
   * Changing the appearance of typography
   * Translating text into other languages
   * Aligning, layering, and formatting objects
   * Inserting pictures and graphics
   * Working with tables
   * Creating master pages
   * Using Design Checker to fix design issues
   * Preparing publications for the web
   * Performing mail merges
   * Preparing publications for commercial print
   * Previewing and printing publications
   * Sharing publications with Pack and Go
