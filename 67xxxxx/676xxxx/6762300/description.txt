  PDF to All Document Converter v5.4  
                                      
      Release Date : 2011-22-10       
          Nfo Date : 2011-22-10       


  Ailt PDF to All Document Converter is an application which     
  you can use to convert PDF files to any file format like       
  Word, Excel, PowerPoint, SWF, image etc. It can support        
  comprehensive formats: DOC, DOCX, DOCM, RTF, XLS, XLSX,        
  XLSM, PPT, PPTX, PPTM, SWF, TXT, HTM, JPG, JPEG, BMP, EMF,     
  WMF, TIF, GIF, PNG, PCX, ICO, J2K, JP2 etc. It does not        
  need Adobe Acrobat software support. The original layout,      
  text, images, vector drawings of your PDF files will be        
  preserved.
