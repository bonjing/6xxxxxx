WinRAR is easy to use file compression and decompression program that effectively integrates with Windows Explorer. RAR compressed files in most cases up to 15 percent better than traditional ZIP-files. Efficient packaging in addition to WinRAR includes among others, the opportunity to file commentary, on the ZIP format compression and decompression, support for long file names, the possibility of corrupt files, repair, file sharing, multiple packages and support self-extracting files (SFX),.

RAR and ZIP formats, in addition to WinRAR supports the following formats: 7Z, ACE, ARJ, BZ2, CAB, GZ, ISO, JAR, LZH, TAR, Uue, Z
