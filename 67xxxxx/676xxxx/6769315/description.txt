This is Yin Yang - X-Change Alternative from Peach Princess. It's 18+ with gender swapping and futanari themes. The game is in English and pre-cracked, so there's no need to mess around with duplicate files.



Programs needed: 

WinRar, available for free from here; 
                  
http://www.rarlab.com/

Daemon Tools Lite, available for free from here;
 
http://www.daemon-tools.cc/eng/products/dtLite




Installation instructions:

Right click on YinYang.part1 and select Extract Files, then OK. This will automatically extract part 1 through 8.

You should end up with an image file named YinYang inside the new folder.

Open Daemon Tools Lite and click Add Image on the left-hand side. Find the directory where the image file is located and select it.

Right click the image that appears at the top of the window, then mouse over Mount, then select the F: drive.

The Auto-play window should open, click Run setup.exe

Follow the Yin Yang setup instructions.

It is not required to keep the image mounted afterward.

Seed and enjoy!
