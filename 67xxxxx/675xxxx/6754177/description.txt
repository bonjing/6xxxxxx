Title: Age of Mythology: The Titans 1.03 [Gold] [ENG/RUS] (+third party updating), Type of publication: license, Genre: RTS, Developer: Ensemble Studios Corp., Year: 2003 Platform: PC, Version: 1.03 Language: Eng / Rus (New Disk LTD), System Requirements: 1.6 GHz CPU, 256 MB RAM, 128 MB video card + 3D, 2 GB HDD free space, Win2000/WinXP or later

 -- English language --
This game is one of the best RTS games of old, known to many established Ensemble Studios Corp. This package without any problems working with such resources as GameRanger and Garena. Also there is an automatic update of the product. After installing the updates, greatly expanding the number of maps in comparison with the standard package AoMTT.
The original disk image is in the root distribution. In the distribution also includes two third-party updating. The first package - it's Nottud's Ultimate Superpack, and the second package created by me based on the code Nottud's Ultimate Superpack and Pete's AOMx User Interface packages, which I have added additional functionality and fixed some bugs and errors. Additional information can be found in the files of the info.txt directory only in Russian. English info_eng.txt file in the root directory.
Installation: Mount the image of the original game to a virtual disk and install, the game first, then two consecutive updates at will, they are in separate folders in the root.
