Title: Two Thrones
Genre: strategy / historical / real-time
Released: 10th February 2004
Developer: Paradox Interactive
Publisher: Paradox Interactive
Description:
From Joan of Arc to Richard III - Two Thrones takes place in the years between 1337 and 1490, in a time when the nobility and the church held the power. Noble families in France and England are engaged in an epic war for the control of the Two Thrones of the kingdoms, a battle that will define the future of medieval Europe.

Bonus content: manual.

How to play:
1) Download the game;
2) Mount or burn disc image;
3) Run Setup.exe to install (it will start automatically if autorun is enabled);
4) Launch the game from the main executable or shortcuts (no crack needed).

Warning!
Windows 7/Vista users: before installing the game, set the Compatibility mode for Setup.exe to Windows XP (Service Pack 3).
