Changes: This is an updated version of 4MLinux-2.3-server-edition, using 4MLinux-3.0 as a core system. The servers controlled by the inetd daemon are: FTP, HTTP, SFTP, SSH, and telnet.... A proxy server (Polipo 1.0.4.1) and JAP (Java Anon Proxy 00.16.001) are also available. The programming environment includes the CGI version of PHP 5.3.8, the minimal version of Perl 5.14.1, and SQLite 3.7.8. The main security components are 4MLinux Firewall (based on iptables 1.4.12.1) and F-Prot Antivirus
