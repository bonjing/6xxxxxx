Customer Review

The Eclipse IDE is a great web development platform. Eclipse supports a wide variety of free plug-ins for PHP, Javascript, SQL, and many, many more. It is easy to use and has many features like auto completion, syntax checking, code formatters, and a built-in web browser. It is highly configurable for to meet a wide variety of needs. Being java-based, it runs on any OS platform that supports a JVM.


NOTES:
This does not come with JDK needed to run eclipse. You will need to download that from here: http://www.oracle.com/technetwork/java/javase/downloads/java-se-jdk-7-download-432154.html
