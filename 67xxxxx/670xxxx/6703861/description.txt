Game Notes
----------
Tief im Dschungel am riesigen Amazonas liegt, so sagt man, das
sagenumwobene Eldorado.
Gehen Sie auf Schatzsuche, sammeln Sie Diamanten, Gold und Rubine und lueften Sie das Geheimnis um diese einst so maechtige Stadt.
Vertauschen Sie benachbarte Felder, indem Sie diese anklicken und bilden Sie dabei eine Reihe aus drei oder mehr gleichen Steinen, damit sie sich aufloesen. Erspielen Sie die sieben heiligen Amulette, die den magischen Schluessel bilden - nur mit ihm erlangen Sie Zutritt zur geheimnisvollen Stadt Eldorado und seinen unfassbaren Schaetzen! Freuen Sie sich auf eine fantastische Kulisse mit toller Grafik und atmosphaerischer Musik. Dieses Abenteuer werden Sie nie vergessen.

Features:
---------
- 65 einzigartige Levels
- Finden Sie verschollene Artefakte
- Match-3-Abenteuer mit nie da gewesenen Features
- Einfacher Einstieg fuer Anfaenger, Steuerung per Maus
- Fuer zwischendurch oder stundenlangen Spielspass
