Changes in DVDFab 8.1.2.5 Qt Beta (September 29, 2011):

    * New: Updated language files.
    * New: Added "3D Format" and "to TS" in "DVD Ripper", "Blu-ray Ripper", "Blu-ray 3D Ripper" and "Video Converter" options.
    * New: Show newly created device in left navigation bar automatically, which works with new Profile Editor.

DVD Copy:

    * New: Added support for a new copy protection as found on "Transformers: Dark of the Moon" (US).
    * New: Added support for a new copy protection as found on "Pirates of the Caribbean: On Stranger Tides" (Germany).
    * Note: HD Decrypter version doesn't support them for now.
    * Fix: Layer break position is not written when using Clone mode.
Blu-ray Copy:
    * New: Added support for latest BD+ protection as found on "Star Wars".
Blu-ray 3D Ripper:
    * Fix: A freeze problem at 71% when converting "3-D Sex and Zen Extreme Ecstasy".
    * Fix: A crash problem when cancelling conversion in certain cases.
