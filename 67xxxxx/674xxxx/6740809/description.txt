CCleaner is system optimization and cleaning tool. The program can easily remove junk files from your computer, which in turn will help Windows run faster.

Currently CCleaner is able to monitor and clean up the example. the following applications: Internet Explorer, Firefox, Windows OS, Windows Registry, and this in addition to several other applications.
