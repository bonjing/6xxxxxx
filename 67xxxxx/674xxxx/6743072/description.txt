Title: Journey to the Moon
Genre: adventure / puzzle / sci-fi
Released: 16th August 2005
Developer: Kheops Studio
Publisher: The Adventure Company
Description:
Journey to the Moon is a point-and-click adventure inspired by Jules Verne's two novels From the Earth to the Moon and Around the Moon. But unlike those two books, the main character actually makes it to the moon surface, although his two companions do not survive the trip.

Bonus content: music, wallpapers.

How to play:
1) Download the game;
2) Mount or burn disc images;
3) Run Install.exe to install (it will start automatically if autorun is enabled);
4) Launch the game from the main executable or using the desktop & start menu shortcuts.
