Randomly generated dungeons, items enemies and bosses, you never play the same game twice.

50+ enemy types each with the ability to become \"special\" making them extra deadly but they also drop better loot.

Install instructions:
1. Extract the .zip file
2. Drag the \"the binding of isaac\" folder to the applications folder
3. Start the game

\"Binding_of_Isaac\" starts the game with a loading screen
\"Isaac\" starts the game with no loading screen

Use with ever you want!
