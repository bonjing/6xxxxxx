In order to play, simply extract the .zip file anywhere and run Dune.bat
This will just run a few commands into DosBox to emulate the game on newer systems and is harmless, so feel free to open the bat in notepad and check for yourself.

Save files are stored under /DosBoxPortable/App/DosBox/Dune/DuneCD
