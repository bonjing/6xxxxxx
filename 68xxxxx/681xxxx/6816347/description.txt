This is my first .exe file that I created using Netbeans 7.0.
It is program that encrypts your message to send safely through internet.
Use this program for decryption.
Just paste your message in the Original Text space and select an encryption number, then hit Encrypt pick your encrypted text from the space below.
For Decryption, paste the Encrypted message in the lower space and choose the same number used when encrypting this message. Finally hit decrypt.
