Title: Celtica
Genre: adventure / point&click / 1st-person
Released: 1998
Developer: Artech Studios
Publisher: H+A Entertainment
Description:
Celtica is a point&click adventure set on a tiny and mysterious, but beautiful, green island off the west Irish coast. The game begins with your character on a small hill near the tip of the island. Several exploratory paths are available to you, including a lighthouse, an abandoned church, and a cemetery.

How to play:
1) Download the game;
2) Mount or burn disc images;
3) Run SETUP.EXE to install (it will start automatically if autorun is enabled);
4) Play the game with the disc inserted.
