Title: Clandestiny
Genre: adventure / puzzle-solving / 1st-person
Released: 1996
Developer: Trilobyte Games
Publisher: Virgin Interactive
Description:
Clandestiny is a cartoon puzzle game in which the player wanders around a haunted house solving logic puzzles. Each solved puzzle triggers a brief cartoon to extend the story and also opens another room in the house.

Bonus content: manual, help guide.

How to play:
1) Download the game;
2) Mount or burn disc images;
3) Run SETUP.EXE to install (it will start automatically if autorun is enabled);
4) Play the game with the disc inserted.
