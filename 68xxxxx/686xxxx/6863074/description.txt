About:
"The Pirate Bay Game" The goal of the game is to help the pirate bay collect their escaping pirates.
Move with the cursor buttons the ship over the playing field to collect as much pirates as you can in the given amound of time.
When the time is up you can submid your score to the site and there you can view all scores from all players around the world.


Controls:
Cursor left = Move to left
Cursor right = Move to right
Cursor up = Move up
Cursor down = Move Down
Esc = Exit the game
F4 = Switch between screen modes
F5 = Save the game
F6 = Load the game
F9 = Make a screenshot of the game
