

Ubuntu 11.10 Install Box - an informal assembly of the operating system GNU / Linux Ubuntu, with all the other updates released on November 25, 2011.



When it was created to pursue such goals:



1. Gather operating system the size of one DVD.

2. Add to a variety of programs that meet most of the audience.

3. Closed multimedia codecs (mp3, avi, mkv, dvd, etc.).

4. Drivers for ATI and nVidia.

5. Fonts Microsoft (Times New Roman, Arial, Verdana, Impack, etc.).

6. Full containment of Russian and Ukrainian languages.

7. Dictionaries of an interpreter Stardict.

8. Global menu to LibreOffice (lo-globalmenu module is not included in the original distribution, as sometimes does not work properly. If this is manifested in you, then just delete it).
