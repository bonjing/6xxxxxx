FEELTHERE - CALL! FOR AIRBUS EVO VOL. 2 FS2004
CALL! for the Airbus EVO Vol.2 is a First Officer simulation. This easy to use software brings great realism into your virtual cockpit making you "feel there".

The Fist officer reads the following checklists:

    * COCKPIT PREP
    * BEFORE START
    * ENGINE START
    * AFTER START
    * TAXI
    * BEFORE TAKEOFF
    * TAKEOFF
    * AFTER TAKEOFF
    * DESCENT
    * APPROACH
    * AFTER LANDING
    * SHUTDOWN

After reading each item it will wait for your action making sure you are properly following the checklist flow.

This also works in FSX,Have fun Guys.:+)
