Official Web Site:  http://www.binarybakery.com/menupop.html

MenuPop enables you to pop up the main menu using a hot key or mouse button.

Features:

* Pop up menu at the current mouse location using a hot key or mouse
  button

* Selectable font size (small to super large) menu items

* Selectable menus to show: Apple menu and current application's menu

What's New

* [FIX] Fixed menu anomalies for Word 2011, Excel 2011, Apple Mail 5.x (Lion)

NOTE: Mac App Store version 1.4 has been submitted to Apple. May take a week or so for them to approve it.

System Requirements:

Intel Mac, OS X.6 or better
