Based on Ubuntu, this Remix returns back to the classic GNOME Panel interface that was standard with GNOME 2 and versions of Ubuntu prior to 11.04. 

Features

    * Based on Ubuntu 11.10
    * GNOME Panel interface
    * Additional packages installed by default, including GIMP, Cheese, Eclipse, XChat, etc.
    * Improved Ambiance theme
