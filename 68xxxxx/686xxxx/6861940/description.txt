Calibre is a free and open source app but i thought i would share this because its a great little program



This program will let you convert .MOBI files which kindles use to .PDF which the new Kobo' use.


 
http://calibre-ebook.com/



- Scanned with Kaspersky Internet Security 2012 + AVG Internet Security 2012 -





E-book library management application developed by users of e-books for users of e-books. It has a cornucopia of features divided into the following main categories: 




Library Management



E-book conversion



Syncing to e-book reader devices



Downloading news from the web and converting it into e-book form



Comprehensive e-book viewer



Content server for online access to your book collection
