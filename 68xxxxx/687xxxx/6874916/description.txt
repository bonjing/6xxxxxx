Path Finder is a file browser that combines the familiar Finder interface with the powerful utilities and innovative features.

Features:

* Dual pane file browser

* Tabs to quickly manage multiple locations on your hard drive in one
  window

* Reorder how your files are listed to suit your personal preferences

* View invisible files and navigate inside of Mac OS X packages quickly
  and easily

* Access your frequently used folders, files, and applications with the
  Shelf, and save multiple Shelf setups to suit different activities

* Pause drag-and-drop operations by putting files into the Drop Stack for
  future use

* Create and convert disk images

* Access the OS X command line

* Much, much more!

What's New

* Fixed a bug with the trash menu in Mac OS X 10.7 Lion.

* Fixed a refresh issue after renaming a volume.

* Fixed All Files search item in Lion and other saved searches.

* Fixed an issue with the QuickLook window in Mac OS X 10.7
  Lion.

* Fixed a bug in the subversion plugin.

* Fixed a problem duplicating files with slashes in their name on Mac OS
 X 10.7 Lion.

System Requirements:

Intel, Mac OS X 10.6 or later
