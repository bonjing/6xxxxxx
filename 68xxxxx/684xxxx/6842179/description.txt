Title: Eradicator
Genre: action / shooter / sci-fi
Released: 1996
Developer: Accolade
Publisher: Accolade
Description:
Eradicator combines pure action and puzzle solving, with 1st-person and 3rd-person views to switch between. There are 20 different weapons, with the missiles you control after launching perhaps the most unusual. The single-player levels plus five more are all available for multi-player action. Three distinct characters are on offer in this shoot 'em up - choose from Kamchak the lizard or humans Dan and Eleena. The game's path varies depending on which character you choose.

Bonus content: manual.

How to play:
1) Mount or burn disc image;
2) Run Setup.exe to install (it will start automatically if you have autorun enabled);
3) Play the game without the disc using the desktop or start menu shortcuts.
