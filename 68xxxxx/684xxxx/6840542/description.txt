Thanks for downloading ubuntu-reloaded-11.10....This release contains the finest blend of
Ubuntu packages, built off Ubuntu 11.10.

This folder has the following things:
1) A folder containing unetbootin to make bootable usb.

2) Drivers.iso file containing following proprietary drivers:
a) Nvidia (recommended)
b) Broadcom STA drive
c) ATI FGLRX
Steps for offline installation: (search in unity for aptoncd)
APTonCD&gt;Restore&gt;Load&gt;None&gt;browse for Drivers.iso file&gt;enter your password&gt;Restore&gt;Done!...then install the drivers from the Additional Drivers
dialogue.

3) A pdf file illustrating how to enable the offline English dictionary and thesaurus.

4)  Getting Started with Ubuntu - Second Edition and Ubuntuguide.pdf for getting comfortable in using Ubuntu.

5) ubuntu-reloaded-11.10.iso

6) Release notes

7) Videos for introduction and dual booting with windows.
