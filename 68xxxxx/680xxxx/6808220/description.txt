Title: Tonic Trouble
Genre: action / platformer / 3rd-person
Released: 31st August 1999
Developer: Ubisoft Montreal
Publisher: Ubisoft
Description:
Ed, a friendly alien space janitor, accidentally dropped a can of "tonic" on Earth, causing a great ecological disaster. Everything started mutating! Ed is ordered to clean up the place and retrieve the tonic, but someone got hold of it first... And he declared himself Master of Earth! Can Ed survive the attack of the killer vegetables, find the six items he needs to fix Earth, retrieve the Tonic can from Grogh the Hellish and finally get out from Tonic Trouble?

Bonus content: manual, soundtrack, wallpaper.

How to play:
1) Download the game;
2) Mount or burn disc image;
3) Run SETUP.EXE to install (it will start automatically if autorun is enabled);
4) After installation is complete, launch the game using the desktop or start menu shortcuts.

Warning!
I tested the game on Windows 7 x64, but got really bad graphical glitches that made it unplayable. Use an older OS to play it.
