Title: Industry Giant
Genre: strategy / business / managerial
Released: 14th April 1998
Developer: JoWooD Productions
Publisher: Interactive Magic
Description:
A popular simulation game in Germany, Industry Giant has been redesigned for the United States. Now you can build your business with a variety of factories, department stores, and vehicles.

Bonus content: manual, soundtrack, wallpaper.

How to play:
1) Download the game;
2) Mount or burn disc image;
3) Run autorun.exe to install (it will start automatically if autorun is enabled);
4) After installation is complete, check System32 folder for a file called Wing32.dll. If you can't find it, extract Wing32.zip archive to System32 folder;
5) Play the game with the disc inserted.

Warning!
The game doesn't work on Windows 7 x64. I played it using Windows XP SP3 with the compatibility setting "Run in 640 x 480 screen resolution".
