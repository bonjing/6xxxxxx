Resident Evil 5 (Jimbus edition) 
---------------


Features of this version:


  - Can be burnt into a single DVD-5.

  - Every level and weapon is unlocked.

  - Infinite ammo.

  - Contains trainers.

  - It is possible to play through LAN using the included Teknogods' files (they are not infected, though some antivirus say they are).


Important!: it is necessary more than 1 GB of free RAM to install the game (FreeArc uncompression requisites).
