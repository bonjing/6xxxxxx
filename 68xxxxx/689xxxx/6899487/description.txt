The method of encoding that i have used is compatible for all android phones as long you have installed Rockplayer. Movie has been cropped
and resized to achieve maximum full screen Landscape display with overall quality, amazing smooth frame rate with synchronized audio and video 
without any problems. [color=Green]Subtitles[/color] are included, for using them after you repack the rar package you will see there is a subtitle folder. 
Just create one folder in your SD Card for example "movie" and put the video (*.mkv) and the subtitle (*.srt) file of your desireble language in the same folder
and play the movie. To display the correct text on the movie in the subtitle folder you will see a text file "RockPlayer Subtitles Region". To 
choose the region on RockPlayer open the application and go to settings -&gt; "Select Encode Mode" and select the code for your country region.
