Year: 2011
Genre: Arcade
Developer: one_man
Platform: Windows, Linux
Language: English
License: Freeware

System requirements:
- Video adapter and monitor - Super VGA (800 x 600)
- Keyboard
- For Linux: Qt4

Description:
Three favorite game of Brick Game is now on your computer: Tetris, Snake and Arkanoid.

Controls:
up - choose the difficulty
down - select the speed
right, left - select a game
Esc - pause
Enter - start game
