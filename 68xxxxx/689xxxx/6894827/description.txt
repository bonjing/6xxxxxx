KTR911 is disigned to reset trial period of the following products,
running on Microsoft Windows 32-bit and 64-bit versions:

-Kaspersky Pure
-Kaspersky Anti-Virus 2010
-Kaspersky Anti-Virus 2011
-Kaspersky Internet Security 2010
-Kaspersky Internet Security 2011


How to reset:

1) Disable Kaspersky Self-Defence
2) Turn Off Kaspersky (Exit Kaspersky from system tray)
3) Run KTR911 and click on [Reset Kaspersky] button
4) In the Activation Wizard select \\\'Activate trial version\\\'
   (internet connection required)
