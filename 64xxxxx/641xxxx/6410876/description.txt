This hack working just for NFS Undercover version 1.0 or version 1.0
 

 First : Open The hack

Secound : Open the game and enjoy ;)


Controls :
- F1 = Free Shop 
- F2 = Unlock all cars and Upgrades
- F3 = Infinite Nitro 
- F4 = Infinite Slowing
- F5 = Ghost Rider ( Cars can't crash you and you can'y hit cars)
- F6 = Easy Hunting
- F7 = Max. Popularity
- F8 = Min. Wanted lvl
- F9 = Max. Wanted lvl
- F10 = Anti-Confiscation
 - F11 = Never Busted 
- F12 =  Win Any Race

                                   That it's all.
