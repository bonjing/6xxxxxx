COMIXWARE SE LITE (MAC VERSION)

Comixware is an interactive storytelling software that marries multimedia presentation and editing in one seamless system that is not only a content builder for author but also an interactive player/editor for viewer. 

MINIMUM SYSTEM REQUIREMENTS FOR MACINTOSH:

- 1.2GHz G4 or Intel processor.
- 512MB of physical RAM.
- 500MB of available disk space.
- 32-bit color monitor capable of 1024 x 768 resolution.
- Mac OS X operating system (10.4 recommended).
- QuickTime driver (version 6.0 to 7.3.0 recommended). 
- Internet connection. 

INSTALL NOTE

1. Place Comixware Assets folder at the root of Macintosh HD. 
2. Put Comixware SE folder in Applications directory. 
3. Read QuickTime.txt for driver requirement.
4. Launch the program and you are in! No registration or license needed ;) 
5. Visit www.comixware.com for more info. 

That's it, enjoy!
