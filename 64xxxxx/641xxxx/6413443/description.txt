Loch Journal is a very small and lite blogging software created by Mark Smith. It is usually used for Live Journal

Very neat piece of work but the interface is *19 Century* style. So it can be a bit boring.


The blogs are needed to be written in html codes. So if you want to add paragraphs, headings or anything other than plain text, you better use some html codes. This might help:
 
http://www.make-a-web-site.com/

It has many useful html commands. Best of Luck
