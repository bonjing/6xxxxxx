How to activate a trial version of installed application?

open the main activation window
in the left lower corner of the main application window, click License
in the License management window, click Activate the application with a new license
in the Kaspersky Anti-Virus 2011 Activation Wizard window, select Activate trial version
wait until the Activation Wizard establishes connection with the Kaspersky Lab server and activates a trial version
click the Close button in the License management window
close the main application window
