+------------------------------------------------+
|Dark Dimensions: City of Fog Collector's Edition|
+------------------------------------------------+

+----------------+
|Game Description|
+----------------+

Silvertown was swallowed by fog and disappeared from the face of the earth 100 years ago. Search for a town consumed by fog! Unlock forbidden doors and uncover mysterious items in this thrilling Hidden Object Puzzle Adventure game! 

+----------------------------------+
|Instructions / System Requirements|
+----------------------------------+

No install is required for this portable version, but it must be registered with a key. Extract the ZIP file and create a shortcut to the executable (EXE) to launch the game, then follow the instructions in the text file to register. The Included keygen is NOT malicious or a virus, and any software that claims otherwise is throwing up a false positive.

- OS: Windows XP/Vista/7

- CPU: 1.6 GHz

- RAM: 1024 MB

- DirectX 9.0c or later
