this is highly compressed version of god of war 1 for pc using pcsx2 emulator..

Step no:01
firstly go to god of war\PCSX2-r2711_0.9.6_and_0.9.7_and_Plugins\0.9.6" directory.
then run "pcsx2-r2186_VC10.exe".
Note : Configuration Automatically Opens
If Not Then Apply Following Step

Step No : 02
1.Select "Config"
2.Select "Configure"

Step No : 03 

1.Click "Set Bios Directory"
2.Select the bios folder(provided besides iso)
3.Click "Set Plugins Directory"
4.Select the Plugins folder besides the iso
5.Goto "Cdvdrom"
6.Select "Linuz Iso CDVD 0.8.0"
7.make all the configurations as shown in "config.jpeg"
8.Select Final "OK" 

Step No:04
1."Return to Emulator"
2.Select "Run" 
3.Select "Execute" 
4. select "god of war.iso"

all done!enjoy

if confused open the page "how-to-play-playstation-2-ps2-games-on-pc" but make configurations as "config.jpeg" only!
