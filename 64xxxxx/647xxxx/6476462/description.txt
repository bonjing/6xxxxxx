Installazione:
1. montate o masterizzate il file immagine.
2. installate il gioco
3. andate nella cartella SKIDROW, copiate la directory binaries e incollatela nella directory di installazione del gioco sovrascrivendo quello esistente.

4. se volete cambiare la lingua del menu e dei sottotitoli al gioco copiate il file BinariesWin32 che si trova nella cartella SKIDROW/language e incollatelo nella directory  \Binaries\Win32\ del gioco;
5. eseguite il file appena copiato, selezionate la lingua desidera e giocate.
