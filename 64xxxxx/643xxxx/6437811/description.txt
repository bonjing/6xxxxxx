Maelstrom, the January 2011 release, illustrates the elegance of merging a complex style infrastructure, with a dynamic, and flexible feature set. Perfect for your portfolio or gallery site.

The theme features animated header background images, configurable to be cross-fading or scrolling, whether in the horizontal or vertical axises. This, in combination with the diverse number of module variations, create a perfectly designed sandbox to develop a beautiful online presence. 

NOTICE: Maelstrom is a Joomla 1.5 Template.

Features and Screenshots

* Animated Background Images
* New RokStories Layout
* Scrolling Module Rows
* Gantry Framework
* 8 Preset Styles
* Fusion with MegaMenu & Splitmenu
* RocketTheme Extensions Styling
* Custom Typography
* Custom JS Form Styling
* 70 Module Positions
* 31 Module Variations - Over 250 Combinations
* FF 3.6, Safari 4, Opera 10.x, IE7-IE8 Compatible
* W3C XHTML 1.0 / CSS 3 Valid

NOTE: Gantry v3.1.6+ is required for Maelstrom to work correctly. For more details on the Gantry Framework, please visit its Dedicated Website.
