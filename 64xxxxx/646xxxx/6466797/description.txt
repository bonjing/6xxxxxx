This IP Changer is a tool to change the IP, which your Tibia Client should connect to. It's for all versions from Tibia 7.1 to Tibia 9.00+.
This IP-changer also has support for MC, and the otserv:// protocol.


Visit 
http://www.otservlist.org/
or 
http://www.tibialist.com/
to find your favourite Open Tibia Server.

Visit 
http://loader.sharpot.net/
for more information about the IP-changer.
