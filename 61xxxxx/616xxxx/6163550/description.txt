	
Windows 7: Windows XP Mode.
Speciaal voor gebruikers van oudere programma's kan Windows 7 Professional, Enterprise en Ultimate worden voorzien van de Windows XP Mode. Hiermee kunt u verouderde programma's waar u (nog) niet zonder kunt, zoals bijvoorbeeld aan een speciaal voor uw bedrijf gemaakt facturatieprogramma, gewoon blijven gebruiken.

Draai oudere applicaties onder Windows 7 dankzij Windows XP Mode
Windows XP Mode voor Windows 7 is een uitbreiding waarmee u oudere programma's toch kunt blijven gebruiken. Hoe dat kan? Simpelweg door een complete versie van Windows XP in een venster te draaien. Dergelijke technologie bestond al in de vorm van Virtual PC 2007. Voor Windows 7 is nu onder de naam Windows XP Mode een compleet voorbereide virtuele machine met daarin opgenomen een legale versie van Windows XP Professional SP3 gratis beschikbaar. Om gebruik te kunnen maken van Windows XP Mode dient u wel Windows 7 Professional, Enterprise of Ultimate hebben draaien, want alleen deze versies ondersteunen Windows XP Mode.

Instaleer eerst de windows xp mode, en daarna de twee updates die er bij zitten.
