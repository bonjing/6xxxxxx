Code Viewer 2 is a source code viewer with syntax highlighting which also reads PDF, Word, Excel and iWork '09 documents. Files can be added through iTunes File Sharing, using the application web interface or checked out and updated from Subversion *repositories.

Review and annotate source code. In portrait or landscape, navigate quickly using function bookmarks and attach notes to source code using the new Annotations feature. When you're done you can email the annotations along with the source code.
