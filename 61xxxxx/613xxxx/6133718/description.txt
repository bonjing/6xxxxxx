Brutal Chess Portable
3d chess

Brutal Chess features full 3D graphics, an advanced particle engine, and several different levels of intelligent
 AI.

Features

Brutal Chess is a 3d chess game with an AI to play against. It's features are relatively spartan,
 making for simple gameplay. It does make use of OpenGL, so it may run slow on slower PCs or those without 
OpenGL graphics support. You can disable reflections and shadows to speed up the game by hitting ESC and 
selecting graphics from the menu.
