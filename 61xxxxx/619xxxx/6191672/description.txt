More Info
Hardware
River Past Video Cleaner is tested to run on any PC with a Pentium III/Celeron CPU of 500 MHz or above. A sound card is required.

It may perform on a PC with lesser spec. However, we will not provide technical support if your PC is below the minimum specification we provided. 
Operation Systems
River Past Video Cleaner supports Windows 98 Second Edition, Windows ME, Windows 2000, Windows XP, and Windows Vista. 
DirectX
River Past Video Cleaner requires DirectX 8.0 or above. We strongly recommend that you always use the latest the latest DX (currently 9.0b) because it provides better stability and performance. 

WMF
For WMA output, you must have Windows Media Format runtime 9 installed. We provide it in a separate download package.
