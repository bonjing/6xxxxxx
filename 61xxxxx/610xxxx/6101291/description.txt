Windows 7 Ultimate X86 X64 and UltraISO

very simple just install ultraISO 

copy the windows file to Desktop 

rightclick and go to ultraISO and then "Add To (FILENAME).ISO"

Now run ultraISO "AS ADMINSTRATOR!"

open the ISO you made 

Next go to Bootable on tabs up top

hit "Make Disk Image"

then follow directions on there and i recommend hitting "Format"

then hit "Write"

NOW ON PC PRESS F12 ON BOOT UP AND CLICK THE USB DRIVE AND IT SHOULD BOOT FROM USB :) 

NOTE DONT TAKE THE USB DRIVE OUT DURRING INSTALLATION!
