Counter-Strike 1.6 NoSteam Patch v43 rev2
Patch by http://botov.net.ua/
Version of our patch: 1.0.3 (10/01/2011)

This patch is designed to update any CS 1.6 to the latest version (at the time of the creation this patch).

Unlike other patches - the patch does not contain any advertising in the game, and this patch could be called the as second revision of patch v43, because it contains many fixes that were included in Project DUDe, which contains the latest fixed version of the patch 43 (eg no "white avatars" in the table with the score and other fixes).

Also, this patch have a multiprotocol: if the client is not connected to the server in normal mode (p48), it will try to introduce as p47 and try again. So you can play up to 99% NoSteam servers (47 and 48 protocol).

If you want to write in chat or in console with Russian or Ukrainian (or any other language that is in your windows) - install the unicode patch.
This patch is beta version. If you CS freeze or crash with this patch - then reinstall the patch without choosing this option.
Important: The ability to write in Russian or Ukrainian (or any other language that is in your windows) for your nickname or in the chat is only available on servers with unicode patch support.

This patch includes the following:
* Patch v43 rev2 to the latest version of Counter-Strike 1.6 NoSteam (build 4554).
* Fully automatic multiprotocol.
* Include Russian or English version of Counter-Strike 1.6.
* Russian or English voice.
* Ability to set a default config of CS 1.6.
* Ability to select the Russian language on the servers by default (for servers with amxmodx).
* Ability to reset the player models to standard.
* Ability to install new (steam) player models.
* Ability to disable the update panel design (for those who does not use a standard design).
* Ability to install unicode (utf8) patch. With this in CS added fully use the multi languages (Russian Chat, Russian nicknames and write in Russian in the console and in other languages).
* Ability to turn off mouse acceleration.
* Working servers search on the Internet (Setti MasterServer).
* Ability to reset the background to the original.
* Ability to reset the game menu to original.
* Ability to set the modified game menu with the added servers Botov.NET.UA.
* Ability to disable folder cstrike_russian (also will be a larger font size).
* No longer be able to replace loading banner from the servers.
* No longer be able to replace game menu from the servers.
* Multiprotocol demo player, you can watch any replays.
