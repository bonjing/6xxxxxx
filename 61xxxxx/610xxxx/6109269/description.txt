This is a Windows 7 installation disk with Service Pack 1 RTM integrated into all of the images.

This disk lets you select which version you want to install during installation (Basic, Home Premium, Ultimate, Pro).

All versions both x86 and x64 are on the same disk, so when you put in the disk you can choose which one you want.

Windows 7 Home Basic SP1 x86
Windows 7 Home Premium SP1 x86
Windows 7 Ultimate SP1 x86
Windows 7 Professional SP1 x86

Windows 7 Home Basic SP1 x64
Windows 7 Home Premium SP1 x64
Windows 7 Ultimate SP1 x64
Windows 7 Professional SP1 x64

Windows 7 Service Pack 1 RTM description
 
----------------------------------------------------
 
All previously publicly released updates for Windows 7

The Windows 7 SP1 package will help keep your PCs and servers on the latest support level, provides ongoing improvements to the Windows Operating System (OS), by including previous updates delivered over Windows Update as well as continuing incremental updates to the Windows 7 platform based on customer and partner feedback, and is easy for organizations to deploy a single set of updates.

Windows 7 SP1 will help you:
- Keep your PCs supported and up-to-date
- Get ongoing updates to the Windows 7 platform
- Easily deploy cumulative updates at a single time
- Meet your users demands for greater business mobility
- Provide a comprehensive set of virtualization innovations
- Provide an easier Service Pack deployment model for better IT efficiency
