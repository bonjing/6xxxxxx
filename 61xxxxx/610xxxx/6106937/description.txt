FreeRapid Downloader - a simple Java-based application for downloading fromDepositfiles,Ifolder, Letitbit and other similar file services.
Key features:
- Support for simultaneous downloads from multiple services
- Downloads using proxy
- Download history
- Tracking clipboard
- Automatic check file's existence on server
- Automatic updates of plugins
- Recognition of simple captchas (CAPTCHA)
- Automatic shutdown
- A lot of custom settings
- Work in OS MS Windows, Linux and MacOS
- A variety of artwork (many styles)
- Simply works!
- Drag & Drop for URL
- A simple API for adding other services (in the form of plug-ins)
