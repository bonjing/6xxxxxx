JavaFX Developers Guide.

The JavaFX platform makes it possible to write applications that can be deployed across devices ranging from cell phones to desktops, with little or no change required. JavaFX applications are written using JavaFX Script, a new and easy-to-use scripting language.

Kim Topleys JavaFX Developers Guide thoroughly covers the JavaFX language and its core libraries and shows you step by step how to develop and deliver JavaFX applications for the desktop and for mobile devices. It provides complete coverage of all aspects of the language.
