This DVD is designed for non Apple hardware (PC only). Support most of hardwawre which run on Intel architecture. Main DVD boot system is SL 10.6.0 with 10.6.2 legacy kerne and Chameleon RC5. In case that you have problem booting this DVD (IOATAFamily.kext kenel panic) please use SL_10.6.6i_IOATAFamily.ppf DVD patch which will fix that problem. Just to let you know this might not work on all machines.

Enjoy in this Snow Leopard.
