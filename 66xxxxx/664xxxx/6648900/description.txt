 The underlying GNU/Linux operating system was upgraded. This release is based on the Ubuntu Natty repository (as of 2011/Sep/01).
 New feature: By default when a GNU/Linux is restored, the udev hardware records (70-persistent-cd.rules and 70-persistent-net.rules under /etc/udev/rules.d/) in the restored OS will be removed. It can be disabled in the expert mode with the option "-irhr".
