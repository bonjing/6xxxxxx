This is a collection of DOS games that have been pre-installed and configured through the DOS frontend DBGL.

DBGL requires Java Runtime Environment 1.5 or higher installed to run, though most will already have this installed.

There are no complicated steps to install this, it just needs to be unzipped to

C:\

...that's it! It should be fine to run after that.
