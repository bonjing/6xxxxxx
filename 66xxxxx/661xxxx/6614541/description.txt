==================Lord Of The Rings Online for Mac OS X====================

=========================================================================== 
Confirmed Working on OS X Lion 10.7.2

Uploaded by TannerHamm

1- open .dmg file
2- drag LOTRO (lord of the rings online) into your applications
3- run lotro 
Note: you are now able to log in but it will not work, the client will be out of date
4- in the window click tools and 'Patch' this can take up to 30 minutes on older machines
5- after it finishes you can close and run lotro (you mat need to exit and reopen the application after patching)
