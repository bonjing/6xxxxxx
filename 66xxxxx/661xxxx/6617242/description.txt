######################

This is the full version of Civilization II: Multiplayer Gold Edition, movies included. It is a port of the Windows version using a special wrapper for Wine. 

-Single .app file, click and run.

-You'll need a newer Intel Mac to play it.

-Minimum OS is Leopard.

-You may select 'Config' in the launch menu to run the map editor.

## INSTALLING ########


Decompress 7z file and run file from any location.

You can download 'The Unarchiver' from http://wakaba.c3.cx/s/apps/unarchiver.html or the Mac App Store.

It's open source and of course free.
