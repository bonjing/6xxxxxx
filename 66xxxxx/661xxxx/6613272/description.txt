######################


The original Unreal Tournament was a blasting success when it was released way back in 1999. This is the \'Game of the Year\' edition.



## INFO ##############


* An Intel Mac is required to play this game.

* This is a Cider port of the PC version, and it plays beautifully.

* Tested on a 13\" Macbook Pro mid 2010 version running Lion.

* Updated version of Leopard is minimum.

* UT version is 436

* Music is included



## INSTALLING ########


Decompress 7z file and run file from any location.

You can download \'The Unarchiver\' from http://wakaba.c3.cx/s/apps/unarchiver.html or the Mac App Store.

It\'s open source and of course free.
