GNUGuitarINUX is a GNU/Linux Live CD with a Realtime kernel aiming to be used as an electric guitar pedal effect.

Features

Based on Debian Squeeze
Custom RT kernel - linux-image-2.6.33.7-rt29-gnuguitarinux_2.6.33.7-rt29

Guitarix v0.11.1
SooperLooper v1.6.14
Tuxguitar v1.2
Jackd2
Jack Capture v0.9.5
Hydrogen v0.9.4
