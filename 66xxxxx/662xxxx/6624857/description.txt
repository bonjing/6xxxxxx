These countries are included in this (sample only) filter file
- ready for use in utorrent software etc (ipfilter.dat text file format)
use notepad to edit the filter as required - ranges and country descriptions
shown in data file to the right too.

I will give out a kit so you can generate your own list of ideas
to include or exclude, and to maintain updates to this list from
the net, using the freely available GeoIPCountryCSV.zip 
imported into FoxPro (available here) - cheers! Michael Rizzo Chessman
