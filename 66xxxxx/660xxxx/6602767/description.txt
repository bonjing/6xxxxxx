Elevator simulator

description:
Simulator elevator in an apartment building. You can walk around the building, call the elevator, ride the elevator between floors of different, clicking on different buttons in the elevator, and you can get into the elevator shaft ... In general, all in real life. There are two versions of the game - for x86 and x64 platforms.

System requirements:
Windows XP/vista/7
processor 2GHz
1GB RAM
64MB video

1. If you have more than 2 GB of RAM and is 64-bit Windows, then download the file
skyscraper20alpha7-x64.exe. If you have a 32-bit file download scraper20alpha7.exe.
2. Install
3. Run the shortcut on your desktop Skyscraper
4. play
