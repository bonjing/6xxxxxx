Garmin BaseCamp Special v3.2.2


This version disable online check for the 25 digits unlock code generate by Jetmouse keygen and works with the latest Garmin maps releases.

Instructions for use:
- Install BaseCamp 3.2.2 (get it fromwebsite)
- Replace the original BaseCamp.exe with the patched one

The file downloaded from Garmin's website is an update, but if you haven't a previous version already installed on the PC simply decompress the BaseCamp_322.exe with Winrar in a temporary folder and launch BCMAIN.msi

Have fun!
