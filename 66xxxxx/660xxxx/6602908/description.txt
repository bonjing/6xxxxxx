Rosetta Stone 3.4.5:

Information:

Rosetta Stone is a language learning program to learn any language fast, and easily. With a large collection of audio versions, you can learn almost every major language with ease, almost like your own native language. Learning can be hard, but with creative actives and personal goals, you can accomplish it soon that you think!

Simple to install, and easy interface will make you feel more successful! Home school or Personal, you can learn a language in the comfort of your own home or just about anywhere!

Requirements: (Minimum)

Windows 2000, XP or Vista
512 MB RAM
1 GHz or faster processor speed
600 MB free hard-drive space (per level)
16-bit sound card
800 x 600 display resolution (1024 x 768 recommended)
Speech recognition feature requires a headset microphone (USB recommended)
16X CD-ROM Drive (for installation)

Installation:

Install Rosetta Stone with the Installation, and set to default settings. Then, grab the (RosettaStoneVersion3.exe) and place it into the installation directory:

*This is for Windows Operating Systems Only!
C:/Program Files/Rosetta Stone/Rosetta Stone Version 3         32-Bit
C;/Program Files (x86)/Rosetta Stone/Rosetta Stone Version 3   64-Bit

Now you just add a CD-ROM of any language and open Rosetta Stone to use it.
