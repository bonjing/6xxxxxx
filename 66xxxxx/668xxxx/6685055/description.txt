Garmin City Navigator Australia and New Zealand NT 2012.30 for Mapsource


The tool Unlock Mapset.exe doesn't work on this release, so installation way must be changed.


Installation:
		
Install Mapsource v6.16.3, apply the patch (search TPB).

Unrar the CNAUSNZNT 2012.30_MS.rar file in C:\Garmin\CNAUSNZNT2012.30 (NOT on the desktop !!)

Run (as Administrator) the Install.bat file

Start Mapsource, select the new map


To uninstall, run Uninstall.bat file


Enjoy !!
