winamp2andfriends-r1

an old version of winamp (2.91, release 20030416) and skins and plugins for it 
i've accumulated since 2 came out. any comparison i've done to other 2+ 
(but &lt;3) versions was many moons ago but i remember concluding that this was 
the last huge update to 2 where functionality was concerned and the last one 
before AOL started (completely) shitting all over it.

many may be useless for you but here are the ones i still know to work well/use 
regularly/have some tips for:

albumlist143 hasn't crashed for me unlike albumlist2+

analogvu analog and digital vu vis

audioscrobbler last.fm support

dlitez external par can triggering, no idea if it'll still work with current 
windows device protection

gen_(s)joy/mt_joyc used one of these quite a while ago for a simple ps1 pad 
control on LPT interface, again might not work with current windows device 
protection

G-Force_22 g force gforce won't nag, fucking sweet ass visualizer

WhiteCap_412 whitecap also won't nag, also fucking sweet ass visualizer

InFlac-1.1.2 works with scrobbler, doesn't support new 1.2.1 encoded 24b 
seeking, for that use the in_flac in flac-1.2.1-win.zip. which doesn't support 
the scrobbler. anybody in the comments here want to hack the 1.2.1 decoder into 
the plug with winamp library support or point me in the direction of how to do 
it?

lucidamp35 transparency with albumlist support

shoutcast-dsp-1-9-0 don't know if there's a newer one that works with winamp2

Wawi winamp web interface, if you're going to use this publicly do so over ssh

Winamp_Control_Executables pretty self explanatory

Winamp_KeyController global hotkeys, autoplay/resume, and auto bakup of 
playlist/eq
