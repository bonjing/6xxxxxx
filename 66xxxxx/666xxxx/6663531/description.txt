2011-09-09

4MLinux-3.0-installer released.

All 4MLinux applications have been compiled under Slackware 13.37 (x86/32-bit/SMP):
 - kernel 2.6.39.2,
 - glibc 2.13,
 - gcc 4.5.2.

Have a nice Linux !

If you want to create 4MLinux Live USB, please use the UNetbootin tool which is available at http://unetbootin.sourceforge.net. If you have an older PC (with 64 MB of RAM and 200 MB of free disk space) you should download 4MLinux-3.0-installer.iso. All the other versions are designed for modern PCs with at least 256 MB of RAM.
