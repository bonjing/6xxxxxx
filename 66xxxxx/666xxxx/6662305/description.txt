[How To] Play Dead Island LAN/Internet Tunngle+SmartSteam:

1. Download Dead Island Reloaded the one from extremezone and Install it

2. Install Steam

3. Download SmartSteam

4. Download Update+Coop Fix and extract it to DeadIsland

5. Install Tunngle, make account and join to deadIsland network

6. Run SmartSteam from Steam folder

7. Run Dead Island

8. Play until you come to ACT1

9. Join lobby

10. Enjoy!
