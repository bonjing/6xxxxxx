Pro Evolution Soccer 2011 - PES SMoKE Patch 3.7 Titanium

----------------------------------------------------------------------------------


Installation:

1. Remove any previous patches installed.

2. Unrar files, run (Setup 3.7c.exe) and follow installer until it finishes.

3. When install finishes manager.exe will run, be sure that manager is attached to the exe file, if it shows errors run manager.exe as administrator.

4. extract all files from (kitserver fix) folder to the main kitserver folder i the game install directory.

Important: no konami updates needed.
