Changes in DVDFab 8.1.2.0 Qt (September 15, 2011):

    * New: Updated language files.
    * New: Added "3D Device" in "DVD Ripper", "Blu-ray Ripper", "Blu-ray 3D Ripper" and "Video Converter" options.
    * New: Added support for new glasses-free 3D phones from Sharp, HTC and LG.
    * New: Added interleaved SBS 3D output, which can be used on iPhone4.
    * New: Added two streams in one file 3D output, which can be used on Fujifilm FinePix REAL 3D V1.
    * New: Brand new internal preview engine.
    * New: Some minor changes and improvements.
    * Fix: A crash problem when changing source in "Settings -&gt; Info" window.
DVD Copy:
    * New: Added support for a new copy protection as found on "Thor" (US). Please note: HD Decryter doesn't support it for now.
    * Fix: The interface layout was wrong for "DVD Copy - Split".
DVD Ripper:
    * New: The default profile for AVI was changed to 2GB+ which allows output file to be larger than 2GB.
    * Fix: A video jerky problem when copying DVD to PS3/iPod/iPhone/iPad with CUDA encoder.
Blu-ray Copy:
    * Info: AACS MKB v26 is out, and it's already supported by old version of DVDFab.
    * Fix: "Convert LPCM/DTS audio to AC3" was always enabled for BD9/BD5 output.
    * Fix: A freeze problem at end of conversion when "Convert LPCM/DTS audio to AC3" option is enabled.
    * Fix: A freeze problem when opening disc with DTS-HD audio (7.1 channels) in certain cases.
    * Fix: A freeze problem when copying Blu-ray with VC-1 video in certain cases.
    * Fix: A freeze problem when copying Blu-ray to BD9/BD5 when subpicture is selected in certain cases.
Blu-ray Ripper:
    * New: The default profile for AVI was changed to 2GB+ which allows output file to be larger than 2GB.
    * Fix: A no sound problem in output for m2ts.passthrough profile.
    * Fix: Shock-wave noise in output when selecting the second DTS-HD Master Audio track.
Video Converter:
    * New: The default profile for AVI was changed to 2GB+ which allows output file to be larger than 2GB.
2D to 3D Converter:
    * New: Improved speed of new 2D to 3D engine, and it's the default engine now.
    * New: Improved the engine to remove most jitters.
    * New: Adjust Gain automatically according to the source.
