The Windows Vista Codec Package is compatible with Windows XP and vista.

The following filetypes are enabled in windows media player 11 and SUPPORTED by the
 
installation of the Vista Codec Package.

amr | mpc | ofr | divx | mka | ape | flac | evo | flv | m4b | mkv | ogg | ogv | ogm

 | rmvb | xvid

All of the above filetypes can be queued to the Windows Media Player Playlist using a right-click.
