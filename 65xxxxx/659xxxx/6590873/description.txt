Universal Maps Downloader v6.71 + Serial
 A very user friendly tool for downloading Maps, released by BEAN.Universal Maps Downloader is a tool that help you to get small tile images from Google Maps, Yahoo maps, MicroSoft Virtual Earth maps, OpenStreet Maps. All downloaded small images are saved on the disk. You can view downloaded maps by Maps Viewer Or you can combine them into a big BMP map by Maps Combiner.
