


Photo Slideshow Creator is a handy tool designed to create a music slideshow with 

beautiful transition effects. The program allows you to add unlimited number of 

photographs and captions, apply themes and much more. Once created, the slideshow 

can be saved as an executable file (EXE), screensaver or video, including full HD.


Photo slideshow software has a convenient, multilingual interface which is easy to 

learn. Create a memorable slideshow of high quality in a few minutes! The new

 version includes ability to burn a slideshow to DVD.
